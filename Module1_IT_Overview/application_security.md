# Application Security

## Common Security Vulnerabilities

### 1 SQL Injection
Attackers insert malicious SQL queries into input fields.

Solution:
Use parameterized queries and input validation.

### 2 Cross Site Scripting (XSS)
Attackers inject malicious scripts into web pages.

Solution:
Validate user inputs and escape HTML characters.

### 3 Weak Passwords
Users create easy passwords that attackers can guess.

Solution:
Use strong password policies and encryption.
