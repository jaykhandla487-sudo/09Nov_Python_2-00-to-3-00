# Module 1 – IT Industry Overview

## 1. What is a Program?

A program is a set of instructions given to a computer to perform a task.

Example:  
Calculator program adds numbers and shows result.

Basic working of a program:
- Take input
- Process the data
- Show output

---

## 2. What is Programming?

Programming means writing instructions for a computer using a programming language.

Steps in programming:
- Understand the problem
- Plan solution (algorithm or flowchart)
- Write the code
- Run and test the program
- Fix errors (debugging)
- Improve or maintain the program

---

## 3. High Level Language vs Low Level Language

| High Level Language | Low Level Language |
|---|---|
Easy to understand | Hard to understand |
Closer to human language | Closer to machine |
Portable | Hardware dependent |
Examples: Python, Java | Examples: Assembly, Machine code |

---

## 4. Client and Server

Client:
- Device or software that requests data
- Example: Web browser

Server:
- Computer that provides data or services
- Example: Web server hosting website

---

## 5. TCP/IP Model

TCP/IP is used for communication over the internet.

Layers:
- Application Layer
- Transport Layer
- Internet Layer
- Network Access Layer

These layers help devices communicate with each other.

---

## 6. HTTP vs HTTPS

| HTTP | HTTPS |
|---|---|
Not secure | Secure |
No encryption | Uses encryption |
Port 80 | Port 443 |
Data not protected | Data protected |

---

## 7. System Software vs Application Software

| System Software | Application Software |
|---|---|
Controls hardware | Used by users |
Example: Windows OS | Example: Chrome, MS Word |

---

## 8. Version Control (Git)

Version control helps manage changes in code.

Benefits:
- Track changes in project
- Work with team
- Restore old versions
- Manage project history
